# Project-3
This is Project 3 for IS601 at NJIT.The project consists of the creation of a REST API and use of Postman to test the POST, PUT, DELETE and GET request methods. Proof of these method uses are shown in the attached doc. file. As instructed through the Project's details, I obtained this data of MLB Player's from https://people.sc.fsu.edu/~jburkardt/data/csv/csv.html and used this link https://sqlizer.io/#/ to convert my records.
